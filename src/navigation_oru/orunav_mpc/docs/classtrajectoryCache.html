Detailed Description

Cache of steps on the active trajectory.

It is possible to form the preview window using the steps of the active trajectory directly. However, cache gives more flexibility and allows to modify steps when necessary without changing the original trajectory. In particular the steps are modified, when the end is reached, or brake requested.
