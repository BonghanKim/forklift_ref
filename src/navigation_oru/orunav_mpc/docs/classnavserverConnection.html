Detailed Description

Connection to the Navserver (runs on board of a car). This connection is used to obtain the current state of a car. Several strong assumptions are made on parameters of the connection.

Assumption:
Assumes, that the state and data from encoders are sent in the same TCP packet.
Assumption:
Assumes, that the delays in transmission of the TCP packets are small (less than 20 ms).
Assumption:
Assumes, that no more than one message can be available in the reception buffer at the same time.
