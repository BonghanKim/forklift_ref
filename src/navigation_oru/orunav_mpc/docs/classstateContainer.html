Detailed Description

A simple container, which stores states of a car. Estimated state should be used as the initial state of MPC problem.

Assumption:
Assumes that the expected steering angle is always reached. Hence, the steering angle reported by sensors is ignored. If it is not ignored the performance of the controller degrades due to a delay in sensor reports. The duration of the delay is about 3 sampling periods (3*60 ms). Computation of an error with 180ms delay does not make any improvement, most likely due to the work of the onboard controller.
